Usage Guide


Analysis View

A summary table of input files, key configuration options and the date of running the CoverView analysis. In addition, the number of failed regions and genes are also provided. (A gene is defined to be failed if it includes at least one failed region.) If a text is too long to be displayed in the table, hover the mouse cursor over it to see the full text in the tooltip.

Genes View

A bar plot displaying per-chromosome read counts and a table at the right side showing the total number of regions and the number of failed regions for each gene. The bar plot presents both on-target and off-target read counts for each chromosome. Clicking on a particular bar in the plot selects/unselects the chromosome. If a chromosome is selected, only the genes located on that chromosome are listed in the table, otherwise all genes are shown.

Failed genes (i.e. those with at least one failed region) are highlighted in red. The table can be filtered to show either passed, failed or all genes by the "Genes" option. The search box helps to find particular genes of interest.

A gene can be selected/unselected by clicking on the row of the table. If a gene is selected, the "Show Regions" and "Show Failed Regions" buttons are visible. With "Show Regions" you can open the selected gene in Regions View where only the regions of the gene will be displayed. Clicking on the "Show Failed Regions" button also redirects to Regions View, but only failed regions of the gene will be listed.

Regions View

A scrollable table of the genomic regions that have been analysed by CoverView. Columns of the table provide the following region-specific metrics: RC, MEDCOV, MINCOV, MEDQCOV, MINQCOV, MAXFLBQ, MAXFLMQ and PASS. For failed regions, metrics values not satisfying quality requirements are highlighted in red.

The table can be filtered to show either passed, failed or all regions by using the "Regions" option. With the "Reads" option, you can switch between displaying the above metrics for forward reads only, reverse reads only or all reads included. The search box helps to find particular regions of interest.

A region can be selected/unselected by clicking on the row of the table. If a region is selected, the "Show Profiles" and "Show Gene" buttons are visible. With "Show Profiles" you can open the selected region in Profiles View. Clicking on the "Show Gene" button redirects to Gene View showing the gene to which the selected region belongs.

Profiles View

An interactive graph of per-base quality profiles aligned with the reference sequence. The visible profiles correspond to the region selected in the left-side panel. The diplayed metrics can be changed with the selector at the top of both y-axes. The following per-base metrics can be selected: COV, QCOV, FLBQ, MEDBQ, FLMQ and MEDMQ. If two metrics are plotted together, the color of each line is the same as of the corresponding metrics selector (blue for the y-axis and red for the y2-axis). Hover the mouse cursor over a point of the plot to see the exact genomic position and metrics value.

Click and drag on the plot to zoom in any section of the profiles. The genomic coordinates of the zoomed region are provided under the graph. You can move upstream and downstream or zoom out with the navigation buttons. It is possible to repeatedly zoom in up to DNA base-level. If the "Axes" option is set to "Normal", the y-axes will have the same scale when you zoom in, otherwise they will be re-scaled to be optimised for the visible section of the profiles.

With the "Reads" option, you can switch between showing the per-based profiles for forward reads only, reverse reads only or all reads included. Quality cut-off values used in the definition of "PASS" requirements can be displayed as dashed horizontal lines by switching on the "Cut-off" option. A horizontal line representing the cut-off value is visible only if the relevant metrics is displayed and have the same color.

Failed regions are highlighted in red in the left-side panel. Clicking on the "Show Region" button will open the selected region in the Region View.
